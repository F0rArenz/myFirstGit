# myFirstGit

A practice repository for learning Git version control as part of the 3813ICT Full Stack Development course.

## Description

This project is a simple Node.js application used to practice core Git workflows: cloning, committing, pushing, branching, and resolving merge conflicts.

## Install

```
git clone https://github.com/F0rArenz/myFirstGit.git
cd myFirstGit
npm install
```

## Run

```
node index.js
```
